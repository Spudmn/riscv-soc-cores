SoC memory map
==============

+-------------------------+------+-------------------------------+
| Range                   | Size | Description                   |
+-------------------------+------+-------------------------------+
| 0x00000000 - 0x00001fff |  8K  | ROM0 (on-chip bootrom)        |
+-------------------------+------+-------------------------------+
| 0x40000000 - 0x40003fff | 16K  | SRAM0                         |
+-------------------------+------+-------------------------------+
| 0x80000000 - 0x807fffff |  8M  | SDRAM0                        |
+-------------------------+------+-------------------------------+
| 0x90000000 - 0x9000001f |  32  | UART0                         |
+-------------------------+------+-------------------------------+
| 0x91000000 - 0x91000007 |   8  | GPIO0                         |
+-------------------------+------+-------------------------------+
